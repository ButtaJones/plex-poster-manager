Alternative Method: Browser Console

If the button above doesn't work, you can manually get the token:

  1. Go to the Plex Web App tab
  2. Press F12 to open Developer Tools
  3. Click on the Console tab
  4. Paste this command and press Enter:
localStorage.getItem('myPlexAccessToken') || window.localStorage.getItem('myPlexAccessToken')

Copy the token (without quotes) and paste it into the launcher.
